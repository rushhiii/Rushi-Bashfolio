### Creating New Themes
Create a new CSS file in the `css/` directory:
```css
/* css/mytheme.css */
:root {
  --terminal-bg: #your-background-color;
  --terminal-text: #your-text-color;
  --terminal-accent: #your-accent-color;
  /* ... more variables */
}
```

Add theme to the theme selector in `main.js`.
